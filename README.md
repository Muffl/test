Berechnung ein pythagoreisches Tripel



Ein pythagoreisches Tripel ist ein Zahlentripel aus drei ganzen Zahlen (a,bc), das die Gleichung

  a² + b² = c² (Satz des Pythagoras)

erfüllt.
Das sind genau die Seitenlängen eines rechtwinkligen Dreiecks (Katheten a und b, Hypotenuse c).

Wenn (a, b, c) keinen gemeinsamen Teiler ausser 1 hat, nennt man es ein primitives pythagoreisches Tripel.

Das Programm ZeichnungenPythogoreischesTriple.py zeichnet ein Dreieck, nach Eingabe der Katheden und 
berechnet die Hypotenuse mit dem Satz des Pythagoras.
